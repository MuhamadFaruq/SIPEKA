# SIPEKA - Personal Finance App

A high-quality personal finance management app built with Flutter, designed to be user-friendly for beginners while powerful for financial literacy.

## Features

- Transaction Management (Income & Expense)
- Budget Tracking
- Wishlist Savings Goals
- Financial Insights & Reports
- Customizable Quick Actions
- Custom Categories

## Tech Stack

- Flutter (Latest Stable)
- Provider (State Management)
- SharedPreferences (Local Storage)
- fl_chart (Charts)
- math_expressions (Calculator)
